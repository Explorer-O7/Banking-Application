# Java Banking Application

A simple console-based banking system in Java.

## Features
- Customer registration
- Open savings/current accounts
- Deposits, withdrawals, transfers
- Transaction history
- Interest application for savings

## How to Run
1. Compile: `javac *.java`
2. Run: `java BankingApp`

## Classes
- `Bank`: Core logic
- `Account` (abstract), `SavingsAccount`, `CurrentAccount`
- `Customer`, `Transaction`
- `BankingApp`: CLI interface
